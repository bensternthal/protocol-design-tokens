## Formatting
- type (string) - linear | radial | repeating-linear | repeating-radial
- degree (value) - 90deg the degree of the gradient defined by the angle.
- colors (string) - color stops in sequential order

```
{
  "name": "greyscale",
  "type": "linear",
  "degree": 90,
  "colors": [
    "#000000",
    "#FFFFFF"
  ]
}
```
